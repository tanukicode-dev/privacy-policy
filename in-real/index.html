プライバシーポリシー

陰リア（in-real） — 開発者: TANUKI CODE

1. はじめに

本プライバシーポリシーは、アプリ「陰リア（in-real）」（以下「本アプリ」）における、ユーザー情報の取り扱いについて説明するものです。

2. 取得する情報と利用目的

情報 目的 備考
位置情報（GPS） GPSミッションの距離判定 端末内でのみ使用。サーバーへの送信はありません
カメラ ミッション関連機能v2以降予定 端末内でのみ使用
マイク ミッション関連機能v2以降予定 端末内でのみ使用
プッシュ通知トークン ミッション通知の配信 ローカル通知のみ。外部サーバーへの送信はありません
端末内ストレージ（AsyncStorage） ミッション進捗・設定の保存 端末内にのみ保存。サーバーへの送信はありません

3. 広告について

本アプリでは、Google AdMob を利用して広告を配信しています。AdMob はパーソナライズド広告の配信のために、広告識別子やデバイス情報を収集する場合があります。

詳細は Google プライバシーポリシー をご確認ください。

4. データの保存場所

本アプリで扱うユーザーデータは、すべてユーザーの端末内（AsyncStorage）にのみ保存されます。開発者が運営するサーバーへのデータ送信は一切行いません。

5. 第三者提供

本アプリは、前述の広告サービス（Google AdMob）を除き、ユーザーの個人情報を第三者に提供することはありません。

6. お子様のプライバシー

本アプリは、13歳未満のお子様を対象としたものではありません。13歳未満のお子様から意図的に個人情報を収集することはありません。

7. プライバシーポリシーの変更

本ポリシーは、必要に応じて改定することがあります。重要な変更がある場合は、本ページにて通知します。
